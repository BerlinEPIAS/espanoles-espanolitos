# espanoles-espanolitos
Servidor de Minecraft de el grupo de amigos de discord de Españoles Españolitos.
## Idea del servidor
El servidor tiene como objetivo ser un SMP por facciones con mecánicas vanilla, pero realmente no lo es porque tiene varios mods añadidos que no rompen mucho la funcionalidad del juego. Recomiendo instalar todos los mods de la carpeta, pues son la mayoría de optimización y preparados para el cliente. Posteriormente podréis ver una sección con más información de los Mods.
## Versión de Minecraft
La versión de Minecraft a usar e instalar es la 1.21.11 Fabric. Se puede encontrar el instalador de Fabric (puede estar anticuado) en "main".
## Mods
Hay una sección en la que podéis acceder a todos los mods que recomiendo instalar y los que tiene el servidor, el único imprescindible en el cliente es Fabric-Api. Recomiendo, por si se encontrasen estos mods anticuados, buscar los mods en CurseForge y descargarlos para Fabric en la versión correspondiente. Algunos de los mods parecen triviales pero pueden ser condicionantes para otros mods, ya que son lo que se llama dependencias, para evitar errores instalad todos.
### Lista de mods
BetterF3
mejora la pantalla F3: más clara, configurable y sin caos visual.

cloth-config
librería para menús de configuración. no hace nada solo, la usan otros mods.

Controlling
gestiona keybinds: buscar teclas, ver conflictos, ordenarlas. pura comodidad.

dynamic-fps
reduce uso de cpu/gpu cuando minimizas el juego o estás AFK. ahorro gratis.

fabric-api
la base de fabric. sin esto, la mayoría de mods no existen.

fabric-carpet
herramientas técnicas y reglas opcionales para debug, optimización y control fino del juego.

factions
añade facciones, claims de territorio, gestión de grupos y protección de zonas.

ferritecore
optimiza uso de RAM. menos consumo, mismo gameplay.

lithium
optimiza lógica interna del juego (IA, ticks, redstone). más TPS, mismo comportamiento.

modmenu
menú para ver y configurar mods desde el juego. calidad de vida.

placeholder-api
sistema de variables dinámicas (placeholders) para que otros mods muestren info.

Searchables
permite buscar en listas y menús (opciones, controles, etc.). pequeño pero útil.

sodium
optimización gráfica brutal. muchos más fps, cero cambios de gameplay.

worldedit
herramienta de edición masiva del mundo (copiar, pegar, rellenar). pensada para admins.

iris
mod de shaders para fabric. permite usar shaders tipo seus, bsl, complementary, etc. se integra muy bien con sodium y no cambia el gameplay, solo mejora gráficos.
